# Students Social Media Addiction Analysis using Power BI

## Project Overview
This project analyzes students' social media addiction patterns using Power BI dashboards and visualizations.

## Objectives
- Identify average daily social media usage
- Analyze impact on academic performance
- Compare addiction levels by age and gender
- Study sleep patterns and mental health effects

## Tools Used
- Power BI
- Excel / CSV Dataset
- Data Cleaning
- Data Visualization

## Dashboard Features
- KPI Cards
- Bar Charts
- Pie Charts
- Line Graphs
- Interactive Filters

## Dataset
The dataset contains:
- Student age
- Gender
- Daily social media usage hours
- Academic scores
- Sleep duration
- Stress levels

## Insights
- Higher social media usage reduces study time
- Students with more than 5 hours/day usage show lower academic performance
- Sleep duration decreases with increased screen time

## Conclusion
The project helps understand how excessive social media usage affects students academically and mentally.
